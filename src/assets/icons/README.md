# Custom Icons Folder

Place your custom PNG icon files here for the dashboard stat cards.

## Required Icons:

- `total-patients.png` - Icon for Total Patients card
- `todays-visits.png` - Icon for Today's Visits card
- `total-revenue.png` - Icon for Total Revenue card
- `pending-bills.png` - Icon for Pending Bills card

## Recommended Specifications:

- **Format:** PNG with transparent background
- **Size:** 48x48 pixels or 64x64 pixels
- **Color:** Full color (the app will handle theming)

After adding your icons, follow the instructions in `DASHBOARD_CUSTOM_ICONS_GUIDE.md` in the root folder.
